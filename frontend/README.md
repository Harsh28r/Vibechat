# OmeTV Frontend

Modern video chat application built with React + Vite

## Setup Instructions

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Create `.env` file:
```
VITE_SOCKET_URL=http://localhost:5000
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
